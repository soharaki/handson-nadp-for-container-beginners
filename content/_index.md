# Introduction

Kubernetes Operatorという仕組みがあります。Kubernetes OperatorとはKubernetes上のアプリケーションの運用を自動化し、構成管理を簡易化するためのソリューションです。

Kubernetes Operatorを単純にKubernetes、あるいはOpenShift上で扱うために必要な知識はそれほど多くはありません。
しかし、単に使用するだけではなく、開発するとなった場合、幅広いバックグラウンドを必要とします。

Kubernetesというプラットフォーム、Kubernetes上で動かすアプリケーションとそのパッケージ管理方法、
Kubernetesの基礎技術であるコンテナ、場合によってはコンテナを構成する細かい技術要素に関しても知識が欠かせません。

この講習では、Kubernetes Operatorという仕組みを知るために、仮想化技術まで遡ってその技術背景を解説します。
